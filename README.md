# Yet Another Custom Character Creator for HD44780 LCD Displays 
#### Click [here](https://sarijen.github.io/lcd-custom-char-creator/) to open the website/page

# Features
- Drag to draw pixels  
- Display color can be set to any HTML color
- All settings are saved as local cookies (but not pixels)
- Responsive enough for small screens
- Quick copy
